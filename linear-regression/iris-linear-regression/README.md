# Iris Linear Regression

## 📌 Project Description
This project applies a linear regression model to analyze the relationship between petal length and petal width in the Iris dataset.

## 🎯 Objective
Predict petal width using petal length.

## 🛠️ Tools & Libraries
- Python
- NumPy
- scikit-learn
- Matplotlib

## 📊 Dataset
- Source: sklearn.datasets.load_iris()
- Features used: petal length
- Target: petal width

## 🚀 Steps
1. Load dataset
2. Select features and target
3. Train linear regression model
4. Visualize results

## 📈 Result
The model demonstrates a strong positive linear relationship between petal length and petal width, indicating that petal length is a strong predictor of petal width.
