# Iris Logistic Regression

## Project Description
This project implements a binary classification model using Logistic Regression on the Iris dataset.

## Objective
The goal is to classify Iris flower samples into two classes based on petal length and petal width.

## Tools and Libraries
- Python
- NumPy
- Matplotlib
- scikit-learn
- Jupyter Notebook

## Dataset
- Source: `sklearn.datasets.load_iris()`
- Selected features:
  - Petal length
  - Petal width
- Target:
  - Binary class label (0 and 1)

## Project Content
This project includes the following parts:

1. Basic theory of logistic regression  
2. Visualization of the sigmoid function  
3. Probability interpretation of logistic regression  
4. Visualization of the logistic loss function  
5. Data preprocessing for binary classification  
6. Model training and prediction  
7. Accuracy evaluation  
8. Probability prediction with `predict_proba()`  
9. Visualization of sample distribution  
10. Decision boundary visualization  

## Model Performance
The Logistic Regression model achieved an accuracy of **0.90** on the test set.

## Visualization
The project includes:
- Sigmoid function plot
- Logistic loss function plot
- Iris sample distribution plot
- Decision boundary plot
